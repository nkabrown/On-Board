### Set Up
Pull down the repository:
```
git clone https://github.com/aphuong/On-Board.git
```

Load required gems:
```
bundle install
```

Setup the database:
```
rake db:setup
```

Start up the server on your local machine:
```
rails s
```

Go to localhost:3000 in your browser.
